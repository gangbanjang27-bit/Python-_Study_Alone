# Chapter1. 파이썬 소개
## - 명령어
\> ls: 현재 폴더 확인하기  
    디렉터리: C:\Users\User  
    현재 폴더에 어떤 파일이 있는지 목록 확인
      
\> explorer  .(명령어 뒤에 마침표 한 칸 띄어서): 탐색기 실행  
    폴더 창 형태로 현재 폴더 내용 확인  

\> cd: 폴더로 이동하기  
    폴더 이름에 띄어쓰기가 포함되어 있을 때는 ""나 ''사용  
  
\> cd ..: 상위 폴더로 돌아가기  

\> python 파일명.py: 파이썬 파일 실행하기  
    명령어를 실행하는 위치와 파이썬 파일이 있는 위치가 일치해야 함  

\> \사용해서 폴더 경로 입력: cd 명령어 여러 번 사용하지 않고 한 번에 폴더 이동 가능  

## - 파이썬 용어
* 문장: 실행할 수 있는 코드의 최소 단위  
    문장 + 문장 + ... + 문장 = 프로그램  

* 표현식: 어떠한 값을 만들어 내는 간단한 코드  
    +, - 는 그것 자체만으로 어떠한 값도 만들어 낼 수 없기 때문에 표현식이라 할 수 없음  

* 키워드: 특별한 의미가 부여된 단어, 에약어  
    <img width="807" height="152" alt="image" src="https://github.com/user-attachments/assets/a71859fe-e882-4b6a-8997-e1e284476164" />  
    파이썬은 대소문 구분. True는 키워드, true는 키워드 아님  

* 식별자: 프로그래밍 언어에서 이름을 붙일 때 사용하는 단어. 주로 변수 또는 함수 이름 등으로 사용
    * 식별자 규칙: 키워드 사용 X  
        * 특수 문자는 언더 바(_)만 허용  
        * 숫자로 시작 X  
        * 공백 포함 X  
        * 의미 없는 단어보다는 의미 있는 단어 사용 권장

* 스네이크 케이스와 캐멀 케이스  
    * 스네이크 케이스: 단어 사이에 언더 바(_) 기호를 붙여 식별자 생성 ex) item_list
    * 캐멀 케이스: 단어들의 첫 글자를 대문자로 만들어 식별자 생성 ex) ItemList  
  
* 식별자 구분하기  
    원래 캐멀 케이스는 '첫 번째 글자를 대문자로 적는다'와 '첫 번째 글자를 소문자로 적는다'로 구분  
    하지만 파이썬 에서는 '첫 번째글자를 소문자로 적는다'라는 캐멀 케이스 사용 X  
    따라서 첫 번째 글자가 소문자라면 무조건 스네이크 케이스  
    반대로 첫 번째 글자가 대문자라면 무조건 캐멀케이스

    * 캐멀 케이스(대문자 시작) -> 클래스
    * 스네이크 케이스(소문자 시작) -> 뒤에 괄호 O -> 함수
    * 스네이크 케이스(소문자 시작) -> 뒤에 괄호 X -> 변수
  
* 주석: 프로그램의 진행에 영향을 주지 않는 코드, 프로그램을 설명하기 위해 사용

* 연산자와 자료
  * 연산자: 스스로 값이 되는 것은 아니고 값 사이에 무언가 기능을 적용할 때 사용하는 것
  * 자료: 리터럴, 어떠한 값 자체를 의미

* 출력: print()
  - print() 함수 뒤에 출력하고 싶은 내용을 쉼표로 연결해서 여러 개 작성 가능  
  \>>> print(52, 273, "Hello") -> 52 273 Hello  

  - 줄바꿈하기
  print() 함수 괄호 안에 아무것도 입력 X  
  \>>> print() -> 빈 줄 출력
